# Bits and Bytes
### Teaching basic Go syntax in a fun way!
##### by Dennis Groenendijk

## Requirements
### Install Raylib game engine
    github.com/gen2brain/raylib-go/raylib

## How to run the game
### 1. Compile the game
    go build -o ./bin/app bits-and-bytes/main.go

### 2. Run the game
    ./bin/app
